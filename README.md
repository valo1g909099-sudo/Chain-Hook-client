# Chain Hook Wallet — Client

A multi-currency OAuth wallet with real-time exchange rate tracking and secure payment authorization.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Copy the environment file and configure it:
   ```bash
   cp .env.example .env.local
   ```

3. Set `VITE_API_BASE_URL` in `.env.local` to point to your backend server.

4. Run the development server:
   ```bash
   npm run dev
   ```
